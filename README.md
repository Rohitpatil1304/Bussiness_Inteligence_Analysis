# Bussiness_Inteligence_Analysis

## About Dataset : 
### 1] Total Records: 11,251  
### 2] Total Columns: 15  

### 3] Attributes  
    a] User_ID	(Unique customer ID)  
    b] Cust_name	(Customer name)  
    c] Product_ID	(Product identifier)  
    d] Gender	(M / F)  
    e] Age Group	(Age category [e.g., 18–25, 26–35])  
    f] Age	(Customer’s exact age)  
    g] Marital_Status	(0 = Single, 1 = Married)  
    h] State	(Customer’s state)  
    i] Zone	(Geographic zone)  
    j] Occupation	(Type of profession)  
    k] Product_Category	(Product type)  
    l] Orders	(Number of orders)  
    m] Amount	(Total purchase amount)  
    n] Status	(Empty column)  
    o] unnamed1	(Empty column)
